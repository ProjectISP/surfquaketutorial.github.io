    Installation

    Installing the core

    Create an environment

    Anaconda

    >> conda create -n myenv python=3.9
>> conda activate myenv

    pip

    Warning, if you are using anaconda, normally is automatically activated the base environment. So first of all deactivate it.

    >>(base) conda deactivate

    >> pip install virtualenv # install in your python the virtualenv package (just in case)
>> python3 -m venv .venv # python<version> -m <virtual-environment-name> <venv_location> 
>> source venv/bin/activate #s ource <venv_location>

    Install the core package

    Once you have created your virtual environmente (either anaconda or pip) you can install surfquake. 

    pip install surfquake

    Installing the GUI

    Open a terminal and type:

    >> git clone https://github.com/rcabdia/SurfQuake.git

    Create an environment

    Anaconda

    >> conda create -n myenv python=3.9
>> conda activate myenv

    pip

    Warning, if you are using anaconda, normally is automatically activated the base environment. So first of all deactivate it.

    >>(base) conda deactivate


    >> pip install virtualenv
>> python<version> -m venv <virtual-environment-name>
>> source venv/bin/activate
    Once the enviroment is surfquake is downloaded from the GitHub repository you have created and activated a new enviroment:

    Install the GUI

    >> cd SurfQuake
>> pip install -r requirements.txt
